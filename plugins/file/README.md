# Plugin File

## Components

| Name      | Description                      | Property | Data Type | Socket Type |
|-----------|----------------------------------|----------|-----------|-------------|
| file      | Represents a file                | filename | string    | input       |
| fs_notify | Triggers if the file was changed | filename | string    | input       |
|           |                                  | trigger  | bool      | output      |
